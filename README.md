## ğŸ”§ Customization

### Colors

All colors are defined as CSS custom properties in the `:root` selector in `css/styles.css`. Modify
these values to change the color scheme:

```css
:root {
    --primary-blue: #1280D7;
    --secondary-blue: #2E93C7;
    --teal-primary: #25B8CA;
    /* ... other colors */
}
```

### Content

- Update company information in `html/index.html`
- Modify services, portfolio items, and contact details as needed
- Replace `assets/icon.svg` with your own logo while maintaining the same filename

### Styling

- All styles are contained in `css/styles.css`
- Organized with clear comments and sections
- Responsive breakpoints can be adjusted in the media queries
